# Automação Jurídica - Backend API

Serviço de backend em Python (FastAPI + PostgreSQL) para automação jurídica, processamento de peças e integração RAG (Retrieval-Augmented Generation).

---

## 🌐 Endereços da Aplicação

* **URL do Frontend (React/Vite)**: [http://localhost:5173](http://localhost:5173)
* **URL do Backend (FastAPI)**: [http://localhost:8000](http://localhost:8000)
* **Documentação Interativa da API (Swagger)**: [http://localhost:8000/docs](http://localhost:8000/docs)

---

## 🚀 Como Iniciar o Backend

1. Certifique-se de que a pasta virtual `venv` está ativa e com as dependências instaladas.
2. Inicie o servidor FastAPI:
   ```bash
   python main.py
   ```
